# Blueprint: 사주 치트키 (Saju Cheatkey) UI/UX 및 엔진 고도화

## 프로젝트 개요
'사주 치트키'는 현대적 감각의 명리학 분석 서비스입니다. 모바일 앱 스타일의 UI와 Gemini AI를 연동한 정밀 사주 분석 엔진을 통해 차별화된 사용자 경험을 제공합니다.

## 주요 변경 사항 (V2)

### 1. 하단 네비게이션 및 탭 개편
- **탭 구성**: [🏠 홈], [🦊 팩폭 사주], [💞 환승 궁합], [✨ 2026 운세]
- **변경점**: '내 프로필' 탭을 제거하고 수익화를 위한 '2026 운세' 탭 신설. 프로필 관리는 각 분석 모달 내에서 처리.

### 2. 정밀 사주 엔진 (`lib/saju.ts`)
- **라이브러리**: `lunar-javascript` 도입
- **보정 로직**: 
  - 한국 표준시(KST) 경도 보정 (-30분) 적용
  - 절기(입춘, 경칩 등) 기준의 정확한 월주 계산
  - 야자시(23:00~00:00) 처리 로직 포함
- **데이터 매핑**: 오행별 색상 매핑 및 일간 기준 MBTI 성향 도출

### 3. Gemini AI 연동 API 라우트
- **사주 분석 (`/api/saju`)**: 8자 데이터를 기반으로 '조선시대 MBTI' 컨셉의 팩폭 리포트 생성
- **환승 궁합 (`/api/gunghap`)**: 두 사람의 사주와 관계를 분석하여 시너지 점수 및 속마음 팩폭 제공
- **오늘의 운세 (`/api/unse`)**: 유저 사주와 오늘의 일진(KST 기준)을 비교하여 하루 운세 가이드 생성

### 4. 수익화 및 프리미엄 UI
- **SajuPreview**: 만세력 미리보기(오행 컬러 적용) 및 분석 내용 잠금(Lock) 처리를 통한 결제 유도
- **UnsePage**: 무료 오늘의 운세 카드 및 990원 신년운세 배너 배치
- **GungHapPreview**: 두 사람의 만세력 비교 UI 및 시너지 점수 노출 예정

## 기술 스택
- **Backend**: Next.js App Router, Gemini 1.5 Flash API
- **Engine**: lunar-javascript
- **UI/UX**: Tailwind CSS, Lucide React, Framer Motion

## 향후 계획
- 결제 시스템(Toss Payments 등) 연동
- 분석 리포트의 카카오톡 공유 기능 강화
- 유저별 분석 기록 저장 (Firebase 연동 고도화)
