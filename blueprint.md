# Blueprint: 사주 치트키 (Saju Cheatkey) UI/UX 개편

## 프로젝트 개요
'사주 치트키'는 현대적 감각의 명리학 분석 서비스입니다. 기존 PC 랜딩 페이지 스타일에서 모바일 앱(App) 스타일로 전면 개편하여 사용자 경험을 최적화합니다.

## 주요 변경 사항

### 1. 하단 네비게이션 바 (Bottom Tab Bar)
- **위치**: 화면 하단 고정 (`fixed bottom-0`)
- **최대 너비**: `max-w-md` (모바일 최적화)
- **구성**:
  - `🏠 홈`: 메인 랜딩 페이지
  - `🔮 사주 분석`: 사주 분석 모달 오픈
  - `💞 환승 궁합`: 궁합 분석 모달 오픈
  - `👤 내 프로필`: 유저 프로필/저장된 데이터 (현재는 로그인 모달 연동 예정)
- **디자인**: 메인 컬러(노란색/고동색) 하이라이트 적용

### 2. 메인 화면 (Home) 개편
- **입력 폼 제거**: 기존 히어로 섹션의 `SajuForm` 제거
- **오늘의 운세 추가**: "오늘의 무료 팩폭 한 줄" 카드 섹션 신규 디자인
- **기존 테마 유지**: 노란색/고동색 컬러 스킴 및 "조선시대 MBTI" 카피라이팅 보존

### 3. 사주 분석 모달 (SajuModal)
- **작동**: 하단 탭 메뉴 클릭 시 노출 (팝업/바텀 시트 스타일)
- **단계별 UI**:
  1. **프로필 선택**: 저장된 사람 목록 (`내 프로필`) 및 `+ 새로운 사람 추가하기` 버튼
  2. **정보 입력**: `+ 새로운 사람 추가하기` 클릭 시 기존 `SajuForm` 노출
- **상태 관리**: `isModalOpen`, `viewType` (selection | form) 관리

## 기술 스택
- **Framework**: Next.js (App Router)
- **Styling**: Tailwind CSS
- **Icons**: Lucide React
- **Animation**: Tailwind CSS Transitions & Framer Motion (Optional)

## 단계별 실행 계획
1. `blueprint.md` 업데이트 (완료)
2. `BottomNav.tsx` 개편: 4개 탭 구성 및 모달 연동 로직 추가
3. `SajuModal.tsx` 신규 생성: 프로필 선택 및 폼 입력 로직 통합
4. `page.tsx` 수정: 히어로 섹션에서 폼 제거 및 오늘의 운세 카드 추가
5. `SajuForm.tsx` 최적화: 모달 내 삽입을 위한 스타일 조정
6. 전체 레이아웃 및 반응형 확인
